# MyDrive

**MyDrive** is a sleek external SSD designed in the iconic Apple AirPort form factor. It connects directly to your **AirPort Extreme (6th gen)** or **AirPort Express (6th gen)** via the USB 2.0 port on the back, expanding storage and functionality while maintaining the classic AirPort aesthetic.

## Features

- AirPort-inspired design with matching dimensions
- USB 2.0 connectivity for seamless integration with AirPort routers
- Compact and functional external SSD housing

## Contents

- **Instructions** – Step-by-step setup and usage guide
- **Purchases** – List of components required to assemble MyDrive

## Getting Started

1. Open the **Instructions** folder and follow the setup guide.
2. Use the **Purchases** folder to gather all the required components.
3. Assemble and connect MyDrive to your AirPort device, and enjoy expanded storage functionality!

## Notes

- MyDrive is compatible with **AirPort Extreme (6th gen)** and **AirPort Express (6th gen)** only.
- Ensure proper USB 2.0 connection to avoid connectivity issues.
